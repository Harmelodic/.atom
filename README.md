# .atom (config)

## Installation

Clone repo:

```
$ git clone https://github.com/Harmelodic/.atom.git ~/.atom/
```

Install packages:

```
$ apm install --packages-file ~/.atom/package.list
```

Open Atom:

```
$ atom
```

## Updating packages

```
$ apm list --installed --bare > ~/.atom/package.list
```
